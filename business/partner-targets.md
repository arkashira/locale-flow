# partner-targets.md

## Partner Integration Roadmap for Locale-Flow

### Target Integrations

1. **Transifex**
   - **Rationale**: A leading localization platform that allows teams to manage translation projects efficiently. Integrating with Transifex can provide users with a seamless way to manage their localization workflows.
   - **Free-tier limits**: Up to 1 project and 1,000 words.
   - **Integration effort**: Medium (M)
   - **Value-add**: Streamlines the translation management process, allowing developers to focus on coding rather than manual localization tasks.

2. **Crowdin**
   - **Rationale**: A collaborative localization platform that supports various file formats and integrates well with development tools. This integration would enhance the localization process for development teams.
   - **Free-tier limits**: 1 project with up to 1,000 strings.
   - **Integration effort**: Medium (M)
   - **Value-add**: Facilitates collaboration between developers and translators, reducing time spent on manual tasks.

3. **Phrase**
   - **Rationale**: A comprehensive localization management platform that offers a user-friendly interface and powerful API for integration. This would enhance the user experience for managing translations.
   - **Free-tier limits**: 1 project with up to 1,000 strings.
   - **Integration effort**: Medium (M)
   - **Value-add**: Simplifies the localization workflow and provides real-time updates on translation status.

4. **Lokalise**
   - **Rationale**: A cloud-based localization platform that focuses on automating the translation process. Integrating with Lokalise can provide users with advanced automation features.
   - **Free-tier limits**: 1 project with up to 1,000 strings.
   - **Integration effort**: Medium (M)
   - **Value-add**: Reduces manual translation efforts and speeds up the localization process.

5. **Google Cloud Translation API**
   - **Rationale**: A powerful translation service that can provide instant translations for various languages. This integration can serve as a fallback or initial translation step before human review.
   - **Free-tier limits**: 500,000 characters per month.
   - **Integration effort**: Small (S)
   - **Value-add**: Automates initial translations, reducing the workload on translators.

6. **Microsoft Translator Text API**
   - **Rationale**: Offers real-time translation services and can be integrated into the localization workflow to provide instant translations.
   - **Free-tier limits**: 2 million characters per month.
   - **Integration effort**: Small (S)
   - **Value-add**: Enhances translation speed and efficiency, allowing for quicker iterations.

7. **Smartling**
   - **Rationale**: A translation management system that provides a comprehensive suite of tools for managing localization projects. This integration can enhance user capabilities in managing translations.
   - **Free-tier limits**: No free tier, but offers a trial.
   - **Integration effort**: Large (L)
   - **Value-add**: Provides advanced features for managing translation workflows, improving overall efficiency.

8. **POEditor**
   - **Rationale**: A collaborative localization platform that supports various file formats and offers an API for integration. This would enhance the localization process for development teams.
   - **Free-tier limits**: 1 project with up to 1,000 strings.
   - **Integration effort**: Medium (M)
   - **Value-add**: Facilitates collaboration and simplifies the localization process.

### Prioritization
- Focus on integrations with affiliate/revenue-share opportunities, such as Smartling and Transifex, to enhance monetization potential while providing value to users.
- Prioritize integrations that offer a balance of free-tier limits and integration effort to maximize user accessibility and ease of implementation.
